# 数据可视化综合展示平台

## 项目简介

这是一个专业的数据可视化综合展示平台，基于Chart.js库开发，实现了多种图表类型的动态展示和交互功能。平台展示了5个不同类型的数据图表，包括时间序列、双轴对比、统计图表和数学函数曲线。

## 🎯 功能特性

### 图表展示
- **时间序列波动图** - 2025年1月1日-2日数值监控，支持折线图/面积图切换
- **销量双轴对比图** - 全国与华东区销量对比，支持双轴折线/柱状图切换  
- **编程语言统计图** - 主流编程语言得票数，支持柱状图/横向柱状图/饼图切换
- **二次函数曲线** - y=x²函数展示，支持曲线图/散点图切换
- **正切函数曲线** - y=tan(x)函数展示，支持曲线图/散点图切换

### 交互功能
- ✅ 一键切换图表类型
- ✅ 锚点导航平滑滚动
- ✅ hover显示详细数据
- ✅ 响应式布局适配
- ✅ 动态数据加载
- ✅ 专业的视觉设计

## 🛠️ 技术栈

- **HTML5** - 语义化标签构建页面结构
- **CSS3** - 现代化样式和动画效果
- **JavaScript ES6+** - 交互逻辑和数据处理
- **Chart.js 4.4.0** - 专业图表库

## 🚀 快速开始

1. 下载或克隆项目文件

## 📊 图表说明

### 1. 时间序列波动图
- **数据范围**: 2025-01-01 00:00 至 2025-01-02 20:00
- **数值区间**: 99.5-103.0
- **关键时间**: 凌晨4点低值、下午16点高值
- **切换模式**: 折线图 ↔ 面积图

### 2. 销量双轴对比图
- **全国销量**: 50-90万件区间
- **华东区销量**: 20-40万件区间
- **旺季特征**: 7-9月为销售高峰
- **切换模式**: 双轴折线 ↔ 柱状图

### 3. 编程语言统计图
- **数据源**: Python 450票、JavaScript 380票、Java 290票等
- **统计维度**: 票数和占比分析
- **切换模式**: 柱状图 ↔ 横向柱状图 ↔ 饼图

### 4. 二次函数曲线
- **函数表达式**: y = x²
- **定义域**: x ∈ [-2, 2]
- **值域**: y ∈ [0, 4]
- **关键点**: (-2,4), (0,0), (2,4)

### 5. 正切函数曲线
- **函数表达式**: y = tan(x)
- **定义域**: x ∈ (-π/2, π/2)
- **值域**: y ∈ (-∞, +∞)
- **特殊处理**: 规避无意义断点
